=== Interactive Generator ===
Contributors: squad T (Deny Ramanda, Rino Dwi Cahyo, Hanasya Shabrina)

== Description ==
Plugin ini dibuat untuk memenuhi kebutuhan Editor Multimedia dan teman-teman Infografik untuk bisa membuat custom interaktif mereka sendiri yang akan ditampilkan pada single post wordpress.
Bahasa yang digunakan adalah html, css, javascript (utamakan vue dan hindari jQuery), juga php.
Konsep plugin ini adalah menyediakan CMS berupa inputan-inputan yang akan menghasilkan shortcode (komponen vue) saat Editor meng-klik button export.
ex: [GallerySlide name="foto-lama" inview="1" data="{'url': 'http://myoui-mina/wp-content/uploads/2021/08/Mu1ylNr.jpg', 'title': 'Myoui Mina Twice','credit': 'Deny Ramanda','text': 'Wanita tercantik abad 2021'}" title="Foto Lama"/]

== Installation ==
Cukup membuat sebuah folder dengan nama "interactive-generator" di root wp-content/plugins/
Kemudian buka localhost/wp-admin/ anda dan aktifkan secara manual di menu Plugins
